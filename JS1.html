js组成

  • javascript 编程语言

    • ECMAScript 标准（语法）

    • DOM Document object model 文档对象模型

    • BOM Browser object model 浏览器对象模型

  • js的使用方式

    • 1.行内
      <button onclick="alert(1)">点我</button>

    • 2.内嵌
      把script标签放在body结束标签上面

    • 3.外链

<script src="js路径"></script> 

    注意：使用src引入的script是不能直接在元素内写js，写了也出不来。

        注意:如果加了src，script中不能执行js了。

  • 通过id的方式去获取元素
    语法:
    document.getElementById('id名')。
    onclick 点击行为（事件）
    把一个触发事件要做的事情放到function内

    document.getElementById(‘id名’).onclick= function( ){ alert(‘哈哈’); }

  • 操作属性:

    • obj.属性

      • 读操作:
        console.log(obj.属性名)

      • 写操作:
        obj.属性名 = 要设置的值

  • 要操作元素的样式:

    • ele.style.某个样式 = ‘值’;

    • ele.className 操作类名

  • 要操作元素内容:

    • input的内容: value
      input.value = ‘哈哈’;

  • 操作别的元素的内容:

    • innerText 操作文本 读操作，只能获得文本

    • innerHTML 操作结构 读操作，获取到某个元素下的所有内容（包括文本、标签）

  • 变量(let var):
    可变的量，把数据存储到自定义的名字中，为了复用，只要使用这个名字，就对应到等号后面的值
    let :

    • 语法 let btn = document.getElementById('btn');

    • 声明变量的时候名字不能重复

    • 不能以数字开头

    • 名字不能用关键字

  • 常量(const)
    不可变的量，把数据存储到自定义的名字中，为了复用，只要使用这个名字，就对应到等号后面的值
    const:

    • 语法：const bg = document.getElementById('bg');
      声明变量的时候名字不能重复
      不能以数字开头
      名字不能用关键字
      console.log('') 打印日志
      console.dir('') 打印详细信息日志
      alert('') 调用浏览器的默认弹框
      alert(”) 调用浏览器默认的弹框,还会阻塞下面的代码,还会自动调用数据的toString方法.
      当alert的时候，输出的结果类型一定是字符串类型。

  • 数据类型:

    • 6种，string(字符串)、number(数字)、boolean(布尔值)、
      null(空)、undefined(未定义)、object(对象)

    • ' ' 或者 "" 是字符串

所有html中的获取的内容信息都是字符串

document.getElementById('ID名');

document.getElementsByClassName(‘类名’);

document.getElementsByTagName(‘标签名’);

document.querySelector( );  获取一个

document.querySelectorAll( ); 获取所有的
